CSS
Методичка

Содержание

CSS (Cascading Style Sheets) - это язык стилей, который позволяет веб-разработчикам контролировать внешний вид веб-страниц. Стили можно применять к различным HTML тегам, чтобы изменять их свойства, такие как цвет, шрифт, размер, отступы и многое другое. В этом руководстве мы рассмотрим основные CSS стили, которые можно применить к различным HTML тегам.
Ниже представлены способы реализации каких-либо HTML элементов с помощью CSS.

Быстрый переход:

  • CSS стили для текста
  • CSS для контейнеров
  • CSS для медии
  • CSS для header-тегов

CSS стили для текста

HTML теги: h1-h6, p, span, ul, ol, li, a, em, i, q, s, b, code, button, input.

1. Сменить шрифт

Чтобы изменить шрифт элемента, используйте CSS-свойство font-family:

    body {
        font-family: 'Roboto', Arial, sans-serif;
    }

Первым указывается шрифт, который вы хотите добавить. После него через запятую идут альтернативные шрифты. Например, если на компьютере у посетителя сайта не установлен шрифт Roboto, браузер использует шрифт Arial.

2. Изменить цвет и размер текста

Цвет текста задаётся с помощью свойства color. Оно принимает значения в различных форматах: можно написать название цвета (red, blue), шестнадцатеричное значение (#ff0000, #00f) или функцию rgb(), чтобы задать цвет с помощью трёх чисел.

    p {
        color: red; /* Красный с помощью слов */
        color: #ff0000; /* Красный с помощью HEX */
        color: rgb(1,0,0); /* Красный с помощью rgb() */
    }

Размер текста меняется свойством font-size. Оно принимает значения в пикселях (px), процентах (%) и других единицах измерения.

    h1 {
        font-size: 24px;
    }

3. Добавить жирность

Свойство font-weight задаёт жирность текста. Оно принимает два вида значений.
Численные: от 100 до 900. Например, 400 — обычный шрифт, а 700 — полужирный.
Ключевые слова: normal — обычный шрифт, bold — полужирный, bolder — жирнее, чем текущий, lighter — менее жирный шрифт по сравнению с текущим. 

    p {
        font-weight: bold; /* Полужирный с помощью ключевого слова */
        font-weight: 700; /* Полужирный с помощью численного значения */
    }

4. Сделать текст курсивным

Для этого есть свойство font-style. По умолчанию у него стоит значение normal, то есть текст обычный, без курсива. Чтобы поменять начертание, используйте:

  • italic — делает текст курсивным;
  • oblique — делает текст наклонным.

    p {
        font-style: italic;
    }

Оба значения создают похожий эффект — текст выглядит курсивным, но есть и разница. Если очень упростить, то italic и oblique немного отличаются в начертании. А ещё не у всех шрифтов есть специальная курсивная версия. В таких случаях oblique делает текст похожим на курсив.

5. Преобразовать текст

Свойство text-transform меняет регистр текста. Оно принимает значения:

  • none — без изменений, значение по умолчанию;
  • uppercase — все буквы становятся прописными;
  • lowercase — все буквы становятся строчными;
  • capitalize — первая буква каждого слова находится в верхнем регистре. 

    p {
        text-transform: uppercase;
    }

6. Добавить подчёркивание

Свойство text-decoration добавляет тексту декоративные эффекты:

  • overline создаёт линию над текстом,
  • line-through делает текст зачёркнутым,
  • underline добавляет подчёркивание.

    p {
        text-decoration: overline;
    }

7. Создать тень

Свойство text-shadow добавляет тексту тень. Оно принимает значение в таком формате:

                    p {
                        text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
                        /*смещение по горизонтали, смещение по вертикали, радиус размытия, цвет тени */
                    }

8. Выровнять текст

Свойство text-align выравнивает текст по горизонтали. Оно принимает значения:

  • left — влево;
  • right — вправо;
  • center — по центру;
  • justify — выравнивание по ширине.

9. Изменить высоту строки и межбуквенное расстояние

Свойство line-height устанавливает высоту строки, указывая множитель относительно размера шрифта. Свойство letter-spacing управляет расстоянием между символами.

    p {
        line-height: 1.5;
        letter-spacing: 2px;
        }

Нюансы

  • У некоторых шрифтов нет жирного и курсивного варианта.
  • Свойство text-shadow делает текст эффектным, но слишком насыщенные тени могут усложнить чтение текста или создать плохой контраст.
  • Слишком большое или маленькое значение line-height и letter-spacing может нарушить интервалы между буквами и строками.
  • При изменении размера шрифта (font-size) следует учитывать, что очень мелкий и крупный текст может плохо отображаться или стать нечитабельным.

CSS стили для контейнеров

HTML теги: div, section, header, footer, main.

1. Ширина и высота контейнера

  • width - устанавливает ширину контейнера.
  • height - устанавливает высоту контейнера.
    .container {
        width: 300px;
        height: 200px;
    }

2. Цвет фона

  • background-color - устанавливает цвет фона контейнера.
    .container {
        background-color: #f0f0f0;
    }

3. Границы контейнера

  • border - устанавливает стиль, толщину и цвет границы контейнера.
    .container {
        border: 2px solid #333;
    }

4. Закругленные углы

  • border-radius - создает закругленные углы у контейнера.
    .container {
        border-radius: 10px;
    }

5. Отступы

  • margin - задает внешние отступы контейнера.
  • padding - устанавливает внутренние отступы контейнера.
    .container {
        margin: 10px;
        padding: 20px;
    }

6. Тень

  • box-shadow - добавляет тень контейнеру.
    .container {
        box-shadow: 2px 2px 5px #888888;
    }

7. Положение и выравнивание

  • position - устанавливает позицию контейнера.
  • text-align - выравнивает текст внутри контейнера.
    .container {
        position: relative;
        text-align: center;
    }

8. Переполнение контента

  • overflow - управляет отображением содержимого, которое не помещается в контейнер.
    .container {
        overflow: scroll;
    }

CSS стили для медии

HTML теги: img, video, svg.

1. Максимальная ширина изображения

  • max-width - устанавливает максимальную ширину изображения, чтобы оно не превышало реальный размер.
                    img {
                        max-width: 100%;
                    }

2. Выравнивание изображений и видео

  • display и margin - выравнивают изображения и видео по центру горизонтально.
    img, video {
        display: block;
        margin: 0 auto;
    }

3. Отзывчивость

  • Используйте медиа-запросы (@media) для настройки стилей в зависимости от размера экрана.
    Например, если вы хотите изменить размер шрифта на маленьких экранах: 
    @media (max-width: 768px) {
        img {
            max-width: 80%;
        }
    }

4. Аспектное соотношение

  • Если вы хотите, чтобы медиа-элементы сохраняли свое аспектное соотношение, используйте
    padding-bottom вместе с position: relative. 
    .video-container {
        position: relative;
        padding-bottom: 56.25%;
    }
    .video {
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
    }

5. Обводка для изображений

  • border - добавляет границу вокруг изображения.
    img {
        border: 1px solid #ccc;
    }

6. Фильтры для изображений

  • filter - позволяет применять различные фильтры к изображению, такие как яркость и насыщенность.
                    img {
                        filter: brightness(150%) saturate(120%);
                    }

CSS стили для head-тегов

HTML теги: link, meta.

1. Стили для ссылок link

  • Стилизация для тега link может быть полезной, если у вас есть различные стили для разных таблиц стилей (CSS).
    link[rel="stylesheet"] {
        display: block;
        margin: 10px 0;
        text-transform: uppercase;
    }

2. Стили для метатегов meta

  • Метатеги могут быть стилизованы для более выразительного отображения, например, метатег meta для определения charset.
    meta[charset="UTF-8"] {
        font-weight: bold;
        color: #333;
    }

Стили для link и meta редко используются и чаще всего стилизация применяется к элементам внутри body для более значимого визуального отображения содержимого страницы.
